Устанавливаем Python
Главный инструмент — язык программирования
🤔
Что такое Python?
Python — это язык программирования. Как иностранный язык, только для компьютера. Ты пишешь команды на Python, компьютер их выполняет. Без Python наш курс не запустится — это самое первое что нужно установить.
1️⃣
Открой браузер и зайди на сайт
Набери в адресной строке: python.org/downloads
Нажми большую жёлтую кнопку «Download Python 3.x.x»
💛
Большая жёлтая кнопка Download Python — нажми её
2️⃣
Запусти установщик
Скачался файл python-3.x.x.exe — кликни по нему два раза.

⚠️ ОЧЕНЬ ВАЖНО! Прежде чем нажать «Install Now» — посмотри вниз окошка. Там есть маленькая галочка «Add Python to PATH». Поставь её! Без этой галочки ничего не заработает.
☑️
Галочка «Add Python to PATH» — обязательно поставь перед установкой!
3️⃣
Проверь что всё получилось
После установки нужно проверить. Открой командную строку:

Windows: нажми Win + R, напиши cmd, нажми Enter
Mac: нажми Cmd + Пробел, напиши terminal, нажми Enter

В чёрном окошке напиши:
Командная строка
python --version
Должно появиться что-то вроде Python 3.12.0 — это значит всё готово! ✅
Если написало ошибку — переустанови Python и не забудь про галочку PATH.

Мост между Python и Minecraft
🌉
Что такое mcpi?
mcpi — это библиотека (набор готовых инструментов) которая соединяет Python и Minecraft. Без неё Python не знает что такое «блоки» и «координаты». С ней — знает всё!

1️⃣
Открой командную строку снова
Windows: Win + R → cmd → Enter
Mac: Cmd + Пробел → terminal → Enter
2️⃣
Введи одну команду
Скопируй и вставь в чёрное окошко, потом нажми Enter:
Командная строка
pip install mcpi
Компьютер начнёт скачивать и устанавливать mcpi. Подожди 30–60 секунд. Когда появится надпись Successfully installed — готово! ✅
💡
Не работает команда pip?
Попробуй вместо pip написать pip3:
Командная строка
pip3 install mcpi
Если и это не помогло — скорее всего при установке Python не поставил галочку PATH. Переустанови Python с галочкой.
03
✏️
Выбираем редактор кода
Где писать программы
📝
Зачем нужен редактор?
Код можно писать в обычном Блокноте — но это неудобно. Специальный редактор подсвечивает код разными цветами, подсказывает ошибки и запускает программы нажатием одной кнопки.

Выбери что тебе подходит:

💙
VS Code — мощный редактор
VS Code используют профессиональные программисты. Он немного сложнее Thonny, но намного мощнее.

📥 Скачай на code.visualstudio.com

После установки открой VS Code → нажми на иконку Extensions (квадратики слева) → найди и установи расширение «Python» от Microsoft.
▶️
Как запустить в VS Code
Открой файл .py → нажми кнопку ▶ Run Python File (треугольник в правом верхнем углу) или Ctrl+F5.
🐍
IDLE — уже установлен вместе с Python
IDLE устанавливается автоматически вместе с Python. Он базовый, но работает сразу без лишних шагов.

Как найти IDLE:
Windows: меню Пуск → ищи «IDLE» или «Python»
Mac: Finder → Applications → Python → IDLE
▶️
Как запустить в IDLE
Открой IDLE → File → New File → напиши код → Run → Run Module (или F5).
🚫
Не пиши код в Word или Блокноте!
Word добавляет невидимые символы и умные кавычки — Python их не понимает и выдаёт ошибки. Только специальный редактор кода или Блокнот с названием файла .py.

Подготавливаем Minecraft
Настройка игры для работы с Python
ℹ️
Какой Minecraft нужен?
Нужен Minecraft: Java Edition — это классическая версия для компьютера. Не Bedrock (Windows 10/11 Edition), не телефон, не консоль — только Java Edition.
✅ Подходит
Minecraft: Java Edition
Скачивается на minecraft.net
Запускается через лаунчер
❌ Не подходит
Minecraft Bedrock Edition
Minecraft на телефоне
Minecraft на Xbox / PlayStation
1️⃣
Установи мод RaspberryJuice
mcpi работает с Minecraft через специальный мод. Его нужно скачать и положить в папку с модами.

📥 Скачай RaspberryJuice на сайте: dev.bukkit.org/projects/raspberryjuice

Скачается файл raspberryjuice-x.x.jar
2️⃣
Или установи Spigot-сервер
Самый простой способ: установи Spigot (локальный сервер для Minecraft) и положи туда RaspberryJuice.

Папка для мода:
Windows: C:\Users\ТвоёИмя\AppData\Roaming\.minecraft\mods
Mac: ~/Library/Application Support/minecraft/mods

Положи скачанный .jar файл в эту папку.
🎮
Более простой вариант — Minecraft Education Edition
Если есть доступ к Minecraft Education Edition — он уже поддерживает Python через встроенные инструменты. Спроси учителя или воспользуйся школьной лицензией.
3️⃣
Запусти Minecraft и войди в мир
Открой Minecraft → Одиночная игра → создай новый мир или открой существующий.

⚠️ Важно: мир должен быть открыт пока ты запускаешь Python-программу. Не в меню, а именно внутри игры.

Как хранить файлы с кодом
Порядок в папках — порядок в голове
📂
Создай специальную папку для курса
Создай на рабочем столе папку python_minecraft. Внутри неё создай папки для каждого модуля:
Структура папок
📁 python_minecraft/ 📁 module1/ 📄 urok_1_1.py 📄 urok_1_2.py ... 📁 module2/ 📄 urok_2_1.py ... 📁 moi_proekty/ 📄 moj_zamok.py
💾
Важные правила для файлов
✅ Имя файла всегда заканчивается на .py — это расширение Python
✅ Не используй пробелы в именах файлов: moj_zamok.py вместо мой замок.py
✅ Сохраняй файл перед запуском — Ctrl+S (Windows) или Cmd+S (Mac)
✅ Каждый урок — отдельный файл
❓ ЧАСТЫЕ ВОПРОСЫ
😤Написал код, нажал Run — ничего не происходит!▼
Скорее всего Minecraft не запущен или ты не в мире (на паузе или в главном меню). Проверь: открыт ли Minecraft, ты внутри мира, не нажата ли пауза (Escape). Потом снова нажми Run.
🔴Появляется ошибка ModuleNotFoundError: No module named 'mcpi'▼
mcpi не установлена. Открой командную строку и введи: pip install mcpi. Подожди пока установится. Если не помогло — попробуй pip3 install mcpi.
🔴Ошибка ConnectionRefusedError или Connection refused▼
Python не может подключиться к Minecraft. Причины: 1) Minecraft не запущен — открой игру и войди в мир. 2) Нет мода RaspberryJuice — установи его. 3) Брандмауэр блокирует соединение — разреши Minecraft в настройках брандмауэра.
🔴Ошибка NameError: name 'Minecraft' is not defined▼
Забыл написать первую строку импорта! Проверь что в самом начале программы есть: from mcpi.minecraft import Minecraft — с заглавной буквой M!
🔴Красные волнистые линии под кодом в редакторе▼
Это не всегда ошибка! Редактор иногда подчёркивает mcpi потому что не знает про эту библиотеку. Просто нажми Run — если программа работает, красные линии можно игнорировать.
❓У меня нет Minecraft. Могу я проходить курс?▼
Да! Модуль 1 и большая часть заданий работают без Minecraft — можно использовать онлайн-редактор Trinket (trinket.io) прямо в браузере. Уроки с mcpi (начиная с урока 1.5) потребуют игру, но остальное доступно без неё.
🌐Можно ли писать код онлайн без установки?▼
Для обычного Python без mcpi — да! Используй trinket.io/python/new или programiz.com/python-programming/online-compiler. Для заданий с mcpi — только на своём компьютере с установленным Minecraft.
