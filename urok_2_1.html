Крипер-учитель
В Minecraft у тебя есть инвентарь — ячейки для хранения предметов. В Python тоже есть такие «ячейки» — они называются переменные. В переменную можно положить число, текст или что угодно!
🧑
Стив
А как создать переменную?
🧙
Крипер-учитель
Очень просто! Придумай имя, поставь = и напиши что хранить. Знак = — это не «равно», это «положи в ячейку»!
Python
имя = "Стив" # в ячейку "имя" кладём слово Стив алмазы = 64 # в ячейку "алмазы" кладём число 64 жизни = 20 # в ячейку "жизни" кладём 20 print(имя) # → Стив print(алмазы) # → 64 print("Жизней:", жизни) # → Жизней: 20
👩
Алекс
А можно изменить значение переменной потом?
🧙
Крипер-учитель
Конечно! Именно поэтому они называются «пере-менные» — они могут меняться! Как предмет в ячейке инвентаря:
Python
алмазы = 64 print(алмазы) # → 64 алмазы = 32 # обменяли часть алмазов print(алмазы) # → 32
Правила имён переменных:
✅ Можно: буквы, цифры, _ (подчёркивание)
✅ Можно начинать с буквы или _
❌ Нельзя начинать с цифры: 2игрок — ошибка!
❌ Нельзя пробелы: мой инвентарь — ошибка!
❌ Нельзя: print, if, for — это зарезервированные слова Python
