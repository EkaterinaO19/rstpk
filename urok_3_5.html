КРИПЕР-УЧИТЕЛЬ
В mcpi каждый блок имеет числовой ID. Вот шпаргалка — запомни хотя бы первые пять!
Популярные ID блоков
0 # AIR — воздух 1 # STONE — камень 2 # GRASS — трава 3 # DIRT — земля 4 # COBBLESTONE — булыжник 5 # WOOD_PLANKS — дерево 9 # WATER — вода 10 # LAVA — лава 56 # DIAMOND_ORE — алмазная руда 57 # DIAMOND_BLOCK — алмазный блок
А вот как определить блок под ногами:
Python + mcpi
from mcpi.minecraft import Minecraft from mcpi import block mc = Minecraft.create() # Получаем позицию Стива pos = mc.player.getPos() # Получаем блок под ногами (y - 1) blok = mc.getBlock(pos.x, pos.y - 1, pos.z) print(f"Блок под ногами: {blok}") # Проверяем тип блока if blok == 2: # 2 = трава mc.postToChat("Стою на траве!") elif blok == 1: # 1 = камень mc.postToChat("Стою на камне!") elif blok == 9: # 9 = вода mc.postToChat("Я в воде!") else: mc.postToChat(f"Неизвестный блок: {blok}")
Совет: getBlock(pos.x, pos.y - 1, pos.z) — блок прямо под ногами. pos.y - 1 потому что Стив стоит над блоком!
