Крипер-учитель
Python — отличный калькулятор! Числа в print() пишутся без кавычек. Python сам вычислит и выведет результат.
Python
print(5 + 3) # → 8 print(4 * 6) # → 24 print(15 // 2) # → 7 (целое деление) print(15 % 2) # → 1 (остаток) print("Пол:", 4 * 4) # → Пол: 16
Числа без кавычек — Python считает. С кавычками — выводит как текст, не считает.
