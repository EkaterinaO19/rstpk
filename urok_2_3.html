До этого наши программы знали все ответы заранее. Но настоящие игры спрашивают игрока! Команда input() делает программу интерактивной — она ждёт, пока ты что-нибудь напечатаешь.
Python — input()
имя = input("Как тебя зовут? ") print("Привет,", имя, "! Добро пожаловать в деревню!")

А если нужно ввести число — например сколько алмазов купить?
🧙
Крипер-учитель
Осторожно! input() всегда возвращает строку str, даже если напечатать число. Чтобы считать как число — нужно обернуть в int():
Python
кол_во = int(input("Сколько алмазов купить? ")) цена = кол_во * 3 # 3 изумруда за алмаз print("Итого:", цена, "изумрудов")
⚠️ Помни: input() всегда возвращает str. Для математики конвертируй: int(input(...)) или float(input(...))
❓ПРОВЕРЬ СЕБЯ
Пользователь вводит число 5. Какой тип у переменной после x = input()?
◇int — Python сам определит
◇str — input() всегда возвращает строку
◇float — если ввели число
◇int или str зависит от ввода

◇Ошибку
Как правильно получить число от пользователя и сохранить в переменную n?
◇n = input()
◇n = int(input())
◇n = str(input())
◇n = number(input())
