КРИПЕР-УЧИТЕЛЬ
Условия можно вкладывать друг в друга! Как матрёшки. Внешний if открывает дверь, внутренний проверяет что за ней.
🧑
СТИВ
Как не запутаться в отступах?
🧙
КРИПЕР-УЧИТЕЛЬ
Каждый уровень вложенности = 4 дополнительных пробела. Внешний if — 4, внутренний — 8, третий уровень — 12!
Python
uroven = 15 est_klyuch = True almazy = 3 if uroven >= 10: print("Уровень достаточный") if est_klyuch: print("Дверь открыта!") if almazy >= 5: print("Купил доспехи!") else: print("Мало алмазов для доспехов") else: print("Нет ключа — дверь заперта") else: print("Уровень слишком низкий")
Правило отступов:
Уровень 1: if ...: → тело начинается с 4 пробелов
Уровень 2:     if ...: → тело с 8 пробелами
Уровень 3:         if ...: → тело с 12 пробелами
💡 Совет Крипера-учителя: Больше 3 уровней вложенности — признак запутанного кода. Попробуй переписать с and или вынести условия в переменные!
